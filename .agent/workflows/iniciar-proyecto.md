# Flujo: Iniciar Proyecto Web Responsive

## Pasos

1. Confirmar el stack tecnológico consultando el skill `monitor-stack`
2. Crear la estructura base de carpetas:
   - `src/` - Código fuente principal
   - `src/componentes/` - Componentes reutilizables
   - `src/paginas/` - Páginas o vistas principales
   - `src/utilidades/` - Funciones helper y logs
   - `src/estilos/` - CSS y estilos
   - `assets/` - Imágenes, iconos, fuentes
   - `docs/` - Documentación del proyecto

// turbo
3. Crear archivo `src/utilidades/logger.js` con sistema de logs

4. Crear archivo `src/estilos/variables.css` con breakpoints responsive

5. Crear `index.html` base con meta viewport para responsive

6. Documentar decisiones en `docs/decisiones.md`
